# Raspberry Spy #

Raspberry Pi project using the Raspberry Pi Camera Module and a PIR sensor to take photos and email them to a designated email.

### Required Packages ###

* python 2.7.8
* python-pip
* python-picamera
* rpi.gpio
* keyring
